
            Global Sales Data Analysis using Power BI

01. 📊 Project Overview
The goal of this project is to perform data analysis on a global sales dataset and create an interactive dashboard using Power BI.
In many businesses, sales data is collected from different countries and products. However, without proper analysis, it is difficult to understand which countries are performing well and how sales change over time.This project solves that problem by analyzing the sales dataset and presenting the results through visual dashboards. The dashboard helps users quickly understand the total sales, total profit, monthly sales trends, and country-wise sales distribution.Using Power BI features such as filters, slicers, and drill-down functionality, users can interact with the dashboard and explore the data in more detail.

The following tools and technologies were used in this project:
Excel  
Used for initial data inspection and preprocessing.
Power Query  
Used for transforming and cleaning the dataset before loading it into Power BI.
Power BI  
Used for data modeling, creating calculated columns, and building interactive dashboards.
Data Visualization Techniques  
Bar charts, line charts, KPI cards, and slicers were used to present insights clearly.

04. 🧹 Data Cleaning & Preparation 
Before performing analysis, the dataset was cleaned and prepared to ensure accuracy and consistency.
The following data preparation steps were performed:
Removed unnecessary columns that were not required for analysis.
Checked for missing values and ensured that the dataset did not contain invalid data.
Converted columns into appropriate data types such as date, text, and numeric values.
Created calculated columns such as profit percentage to understand profitability.
Organized the data into dimension tables and fact tables for better data modeling in Power BI.

05. 🔍 Exploratory Data Analysis (EDA)
Exploratory Data Analysis was performed to understand patterns and relationships within the dataset.
Several visualizations were created in Power BI to analyze the data:
Sales by Country  
A bar chart was created to compare the total sales across different countries. This helps identify which countries contribute the most to total sales.
Monthly Sales Trend  
A line chart was used to analyze how sales change over time. This visualization helps identify trends, seasonal patterns, and growth in sales.
Total Sales KPI  
A KPI card was created to display the overall total sales value.
Total Profit KPI  
Another KPI card was used to show the total profit generated from sales.
Country Slicer  
A slicer was added to allow users to filter the dashboard based on specific countries.

06. 💡 Key Insights
•	United States of America recorded the highest overall sales contribution (around 21%).
•	Canada and France followed closely with strong sales performance.                                                                                  • Mexico showed comparatively lower sales contribution among selected countries.
•	Sales show fluctuations across different months, indicating seasonal trends.
•	Certain months recorded peak sales across multiple countries, suggesting high demand periods.

07. 🚀 Recommendations
Based on the analysis, the following recommendations can be made:
Focus marketing efforts on high-performing countries to maximize sales growth.
Identify low-performing regions and develop strategies to improve sales performance in those areas.
Use monthly sales trend analysis to plan promotional campaigns during peak sales periods.
Continuously monitor sales and profit performance using the Power BI dashboard to support better business decisions.
08. ⚙️ How to Use
The Power BI dashboard created in this project allows users to explore sales data interactively.
Users can view the total sales and total profit using KPI cards.
The bar chart shows sales performance by country.
The line chart displays the monthly sales trend over time.
Using the country slicer, users can filter the dashboard to analyze sales for specific countries.
This interactive dashboard makes it easier to understand business performance and identify trends in the data.
